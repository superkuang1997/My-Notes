# Regex

字符类集

| 元字符 | 说明                                 |
| ------ | ------------------------------------ |
| `\d`   | 任何一个数字字符（等价于`[0-9]`）    |
| `\D`   | 任何一个非数字字符（等价于`[^0-9]`） |
|        |                                      |
|        |                                      |
|        |                                      |
